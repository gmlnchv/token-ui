## Overview

**FontWeightToken** is a specialized variant of the Token component that adds a visual icon to help distinguish font weight values while keeping the tooltip for the exact numeric or named value.

This component implements the `fontWeight` type from the [W3C Design Tokens specification](https://www.designtokens.org/tr/2025.10/format/#font-weight) (ยง8.4), which represents font weights from 1-1000 or named values like "bold" or "normal".

## Spec Compliance

The W3C Design Tokens specification defines `fontWeight` as:
- **Numeric values**: 1-1000 (OpenType wght tag specification)
- **Named values**: `thin`, `extra-light`, `light`, `normal`, `medium`, `semi-bold`, `bold`, `extra-bold`, `black`, `extra-black`

Lower numbers represent lighter weights, and higher numbers represent thicker weights.
