## Overview

**RadiusToken** is a specialized variant of the Token component that adds a visual icon to help distinguish corner radius values while keeping the tooltip for the exact measurement.

## Installation

<Installer packageName="radius-token" />

## Usage

Use `RadiusToken` for border radius values:

<Preview code={`import { RadiusToken } from '@/components/radius-token'

  <div className="flex gap-2 flex-wrap">
    <RadiusToken name="radius-xs" value="2px" />
    <RadiusToken name="radius-sm" value="4px" />
    <RadiusToken name="radius-md" value="8px" />
    <RadiusToken name="radius-lg" value="12px" />
    <RadiusToken name="radius-full" value="999px" />
  </div>`}>
  <div className="flex gap-2 flex-wrap">
    <RadiusToken name="radius-xs" value="2px" />
    <RadiusToken name="radius-sm" value="4px" />
    <RadiusToken name="radius-md" value="8px" />
    <RadiusToken name="radius-lg" value="12px" />
    <RadiusToken name="radius-full" value="999px" />
  </div>
</Preview>
