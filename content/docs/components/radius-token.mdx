## Overview

**RadiusToken** is a specialized variant of the Token component that adds a visual icon to help distinguish corner radius values while keeping the tooltip for the exact measurement.

This component uses the `dimension` type from the [W3C Design Tokens specification](https://www.designtokens.org/tr/2025.10/format/#dimension) (ยง8.2), as border radius values are distance measurements.
