## Overview

**NumberToken** is a specialized variant of the Token component that adds a visual icon to help distinguish numeric values while keeping the tooltip for the exact number.

This component implements the `number` type from the [W3C Design Tokens specification](https://www.designtokens.org/tr/2025.10/format/#number) (ยง8.7), which represents unitless numeric values like line-height multipliers, opacity, or gradient stop positions.
