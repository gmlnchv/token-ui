title: TokenIndicator

import { Token, TokenName } from '@/registry/components/token'
import { TokenIndicator } from '@/registry/components/token-indicator'
import { Blend } from 'lucide-react'

## Overview

**TokenIndicator** is a context-aware mark that sits next to token labels. It reads the token's value from context, applies it as a background by default, and can render any children—icons, glyphs, or extra UI—while remaining fully styleable.

## Installation

<Installer packageName="token-indicator" />

## Usage

Combine `TokenIndicator` with `TokenName` to decorate tokens. Customize shape via `className`, or render custom children while still inheriting the token value as background.

<Preview code={`import { Token, TokenName } from '@/components/token'
import { TokenIndicator } from '@/components/token-indicator'
import { Blend } from 'lucide-react'

  <div className="flex flex-wrap items-center gap-3">
    <Token name="Default" value="#3B82F6">
      <TokenName className="gap-2">
        <TokenIndicator />
        <span>Default</span>
      </TokenName>
    </Token>

    <Token name="Square" value="#3B82F6">
      <TokenName className="gap-2">
        <TokenIndicator className="size-5 rounded-sm" />

        </TokenIndicator>

      </TokenName>
    </Token>
  </div>`}>
