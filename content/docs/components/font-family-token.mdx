## Overview

**FontFamilyToken** is a specialized variant of the Token component that adds a visual icon to help distinguish font family values while keeping the tooltip for the exact font stack.

This component implements the `fontFamily` type from the [W3C Design Tokens specification](https://www.designtokens.org/tr/2025.10/format/#font-family) (ยง8.3), which represents font names or font stacks.
