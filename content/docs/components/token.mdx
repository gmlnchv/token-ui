## Usage

Use the `Token` component to document any design token. The token name appears as a button, and hovering reveals the value. Clicking the token name copies it to the clipboard:

<Preview code={`import { BasicToken as Token } from '@/components/token'

  <div className="flex gap-2 flex-wrap">
    <Token name="spacing-xs" value="4px" />
    <Token name="spacing-sm" value="8px" />
    <Token name="spacing-md" value="16px" />
    <Token name="spacing-lg" value="24px" />
    <Token name="spacing-xl" value="32px" />
  </div>`}>
  <div className="flex gap-2 flex-wrap">
    <Token name="spacing-xs" value="4px" />
    <Token name="spacing-sm" value="8px" />
    <Token name="spacing-md" value="16px" />
    <Token name="spacing-lg" value="24px" />
    <Token name="spacing-xl" value="32px" />
  </div>
</Preview>

### Copy to Clipboard

By default, clicking a token name copies it to the clipboard and displays "Copied" for 2 seconds. You can disable this behavior by setting `copyable={false}`:

<Preview code={`import { BasicToken as Token } from '@/components/token'

  <Token name="brand-primary" value="#3b82f6" copyable={false} />`}>
  <Token name="brand-primary" value="#3b82f6" copyable={false} />
</Preview>

## Composition

The Token component provides **three levels of abstraction**, letting you choose the right balance between convenience and control:

### 1. BasicToken (High-Level)

The simplest approach. Perfect for most use cases. This is the default export.

<Preview code={`import { BasicToken as Token } from '@/components/token'

  <div className="flex gap-2 flex-wrap">
    <Token name="primary" value="#3b82f6" />
    <Token name="secondary" value="#64748b" copyable={false} />
  </div>`}>
  <div className="flex gap-2 flex-wrap">
    <Token name="primary" value="#3b82f6" />
    <Token name="secondary" value="#64748b" copyable={false} />
  </div>
</Preview>

### 2. Token + Custom Children (Mid-Level)

Use when you need custom trigger content but want the standard tooltip behavior. Great for adding icons, indicators, or custom layouts.

<Preview code={`import { Token, TokenName } from '@/components/token'
import { AlertTriangle } from 'lucide-react'

  <Token name="deprecated-color" value="#dc2626">
    <TokenName className="gap-2">
      <AlertTriangle className="size-3.5" />
      <span>deprecated-color</span>
    </TokenName>
  </Token>`}>
  <TokenComposed name="deprecated-color" value="#dc2626">
    <TokenName className="gap-2">
      <AlertTriangle className="size-3.5" />
      <span>deprecated-color</span>
    </TokenName>
  </TokenComposed>
</Preview>

### 3. Primitives (Low-Level)

Full control for advanced use cases. Compose primitives directly when you need completely custom behavior, different tooltip settings, or non-standard interactions.

<Preview code={`import { TokenProvider, TokenRoot, TokenValue } from '@/components/token'
import { TooltipTrigger } from '@/components/ui/tooltip'

  <TokenProvider name="special-token" value="Custom implementation">
    <TokenRoot delayDuration={0}>
      <TooltipTrigger asChild>
        <button className="px-3 py-1 rounded-full bg-orange-100 text-orange-900 text-sm font-medium hover:bg-orange-200">
          Click for details
        </button>
      </TooltipTrigger>
      <TokenValue />
    </TokenRoot>
  </TokenProvider>`}>
  <TokenProvider name="special-token" value="Custom implementation">
    <TokenRoot delayDuration={0}>
      <TooltipTrigger asChild>
        <button className="px-3 py-1 rounded-full bg-orange-100 text-orange-900 text-sm font-medium hover:bg-orange-200">
          Click for details
        </button>
      </TooltipTrigger>
      <TokenValue />
    </TokenRoot>
  </TokenProvider>
</Preview>

### Available Primitives

- **`TokenProvider`** - Context provider for name/value
- **`TokenRoot`** - Tooltip wrapper with configurable delay
- **`TokenName`** - Button trigger with copy functionality
- **`TokenValue`** - Tooltip content displaying the value

Since you own the code after installation, feel free to modify these components, add new props, or create project-specific variants.
