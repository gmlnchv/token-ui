## Overview

**Token** is a general-purpose component for displaying any name–value pair as a tooltip. It provides the foundation for more specialized token components like **ColorToken**, while remaining flexible enough to be extended or composed for custom documentation requirements.

## Installation

<Installer packageName="token" />

## Usage

Use the `Token` component to document any design token. The token name appears as a button, and hovering reveals the value. Clicking the token name copies it to the clipboard:

<Preview code={`import { BasicToken as Token } from '@/components/token'

  <div className="flex gap-2 flex-wrap">
    <Token name="spacing-2" value="4px" />
    <Token name="spacing-4" value="8px" />
    <Token name="spacing-8" value="16px" />
    <Token name="spacing-12" value="24px" />
    <Token name="spacing-16" value="32px" />
  </div>`}>
  <div className="flex gap-2 flex-wrap">
    <Token name="spacing-2" value="4px" />
    <Token name="spacing-4" value="8px" />
    <Token name="spacing-8" value="16px" />
    <Token name="spacing-12" value="24px" />
    <Token name="spacing-16" value="32px" />
  </div>
</Preview>

### Copy to Clipboard

By default, clicking a token name copies it to the clipboard. Feel free to disable this behavior by setting `copyable={false}`:

<Preview code={`import { BasicToken as Token } from '@/components/token'

  <Token name="color-brand-primary" value="#3b82f6" copyable={false} />`}>
  <Token name="color-brand-primary" value="#3b82f6" copyable={false} />
</Preview>

## Composition

The Token component provides **three levels of abstraction**, letting you choose the right balance between convenience and control:

### Pre-Composed

An opinionated, ready-to-use component with sensible defaults. Perfect for common cases where you don't need custom composition.

<Preview code={`import { BasicToken as Token } from '@/components/token'

  <div className="flex gap-2 flex-wrap">
    <Token name="color-primary-600" value="#3b82f6" />
    <Token name="color-neutral-500" value="#64748b" copyable={false} />
  </div>`}>
  <div className="flex gap-2 flex-wrap">
    <Token name="color-primary-600" value="#3b82f6" />
    <Token name="color-neutral-500" value="#64748b" copyable={false} />
  </div>
</Preview>

### Custom Trigger

Use when you need to customize the trigger content with badges, labels, or semantic UI elements while keeping the standard tooltip behavior.

<Preview code={`import { Token, TokenName } from '@/components/token'

  <Token name="color-accent-experimental" value="#a855f7">
    <TokenName className="gap-2">
      <span>color-accent-experimental</span>
      <span className="rounded-full bg-purple-100 px-1.5 py-0.5 text-[10px] font-medium text-purple-700">
        Beta
      </span>
    </TokenName>
  </Token>`}>
  <TokenComposed name="color-accent-experimental" value="#a855f7">
    <TokenName className="gap-2">
      <span>color-accent-experimental</span>
      <span className="rounded-full bg-purple-100 px-1.5 py-0.5 text-[10px] font-medium text-purple-700">
        Beta
      </span>
    </TokenName>
  </TokenComposed>
</Preview>

For visual indicators that display the token's value (like color swatches), you can use [`TokenIndicator`](#tokenindicator) instead.

### Primitives

Maximum flexibility for advanced use cases. Compose primitives directly when you need custom tooltip content, non-standard interactions, or integration with other components.

This example shows a spacing token with a custom-styled tooltip and visual preview—something you can't achieve with the pre-composed components:

<Preview code={`import { TokenProvider, TokenRoot, TokenName, TokenValue } from '@/components/token'

  <TokenProvider name="spacing-8" value="64px">
    <TokenRoot>
      <TokenName className="border-blue-400 bg-blue-500/10 hover:bg-blue-500/20" />
      <TokenValue className="border-blue-400 bg-blue-500/10 px-4 py-3">
        <div 
          className="rounded border-2 border-dashed border-blue-400 bg-blue-500/20 text-blue-500 flex items-center justify-center text-sm font-medium" 
          style={{ width: '64px', height: '64px' }} 
        >
          64px
        </div>
      </TokenValue>
    </TokenRoot>
  </TokenProvider>`}>
  <TokenProvider name="spacing-8" value="64px">
    <TokenRoot>
      <TokenName className="border-blue-400 bg-blue-500/10 hover:bg-blue-500/20" />
      <TokenValue className="border-blue-400 bg-blue-500/10 px-4 py-3">
        <div 
          className="rounded border-2 border-dashed border-blue-400 bg-blue-500/20 text-blue-500 flex items-center justify-center text-sm font-medium" 
          style={{ width: '64px', height: '64px' }} 
        >
          64px
        </div>
      </TokenValue>
    </TokenRoot>
  </TokenProvider>
</Preview>

## API Reference

### BasicToken

A fully composed token component with sensible defaults. Composes `Token` with `TokenName` to provide a button with copy functionality and value tooltip. This is the recommended starting point.

<AutoTypeTable
  name="BasicTokenProps"
  type={`
export type BasicTokenProps = {
  /** The token name */
  name: string
  /** The token value displayed in the tooltip */
  value: string
  /**
   * Whether clicking the token name should copy it to the clipboard.
   * @default true
   */
  copyable?: boolean
}
`}
/>

### Token

Mid-level composition that accepts custom children as the tooltip trigger. Composes `TokenProvider`, `TokenRoot`, and `TokenValue`. Use when you need to customize the trigger beyond the basic button.

<AutoTypeTable
  name="TokenProps"
  type={`
export type TokenProps = {
  /** The token name */
  name: string
  /** The token value displayed in the tooltip */
  value: string
  /** Custom trigger content */
  children?: React.ReactNode
}
`}
/>

### TokenProvider

Context provider that makes token name and value available to all child components.

<AutoTypeTable
  name="TokenProviderProps"
  type={`
export type TokenProviderProps = {
  /** The token name */
  name: string
  /** The token value */
  value: string
  children: React.ReactNode
}
`}
/>

### TokenRoot

Provides the tooltip functionality. Composes `Tooltip` from `shadcn/ui`.
