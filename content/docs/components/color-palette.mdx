## Overview

**ColorPalette** displays labeled color collections as horizontal or vertical stripsâ€”perfect for documenting color ramps, scales, brand palettes, and semantic token sets.

These layouts are intentionally flexible and easy to reshape. You get a pre-composed `ColorPalette` for speed, plus primitives (`ColorSwatch`, `ColorSwatchLabel`, `ColorPaletteLabel`) to build custom layouts like grids or asymmetric compositions.

## API Reference

### ColorPalette

Pre-composed palette component with labels shown by default.

<AutoTypeTable
  name="ColorPaletteProps"
  type={`
export type ColorPaletteProps = {
  /** Array of colors to display */
  colors: string[] | ColorStop[]
  /** Orientation of the palette (default: horizontal) */
  orientation?: 'horizontal' | 'vertical'
  /** Optional label */
  label?: string
  /** Additional class name for the wrapper (label + palette stack) */
  className?: string
}
`}
/>

### ColorSwatch

A single color swatch in a palette.

<AutoTypeTable
  name="ColorSwatchProps"
  type={`
export type ColorSwatchProps = React.HTMLAttributes<HTMLDivElement> & {
  /** Color value (hex, rgb, hsl, etc.) */
  value: string
  /** Optional name/label for the color */
  name?: string
}
`}
/>

### ColorSwatchLabel

Label displayed on hover inside a color swatch.

Inherits all standard div props from the underlying HTML element.

### ColorPaletteLabel

Label displayed above a palette.

Inherits all standard div props from the underlying HTML element.
