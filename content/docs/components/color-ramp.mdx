## Overview

**ColorRamp** displays labeled horizontal strips of colorsâ€”perfect for documenting scales, brand palettes, and semantic ramps.

These layouts are intentionally subjective and easy to reshape. You get a pre-composed `ColorRamp` for speed, plus primitives (`ColorSwatch`, `ColorSwatchLabel`, `ColorRampLabel`) to fully control layout and behavior.

## API

### Props

- **colors** - Array of hex/rgb/hsl strings, or objects with `{ name, value }`
- **label** - Optional text label above the ramp
- **orientation** - `'horizontal'` (default) or `'vertical'`
- **className** - Optional wrapper class for spacing/layout around the ramp
