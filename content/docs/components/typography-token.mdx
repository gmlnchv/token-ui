## Overview

**TypographyToken** is a specialized variant of the Token component that adds a visual icon to help distinguish font-related tokens while keeping the tooltip for precise specs.

This component implements the `typography` composite type from the [W3C Design Tokens specification](https://www.designtokens.org/tr/2025.10/format/#typography) (ยง9.8), which represents typographic styles including font family, size, weight, line height, and letter spacing.

## Usage

Use `TypographyToken` for typeface, size, and line-height values:
