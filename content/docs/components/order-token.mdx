## Overview

**OrderToken** is a specialized variant of the Token component that adds a visual icon to help distinguish stacking context values while using the same tooltip to expose the exact z-index.

## Installation

<Installer packageName="order-token" />

## Usage

Use `OrderToken` for z-index or layering values:

<Preview code={`import { OrderToken } from '@/components/order-token'

  <div className="flex gap-2 flex-wrap">
    <OrderToken name="z-base" value="0" />
    <OrderToken name="z-raised" value="10" />
    <OrderToken name="z-floating" value="50" />
    <OrderToken name="z-overlay" value="100" />
  </div>`}>
  <div className="flex gap-2 flex-wrap">
    <OrderToken name="z-base" value="0" />
    <OrderToken name="z-raised" value="10" />
    <OrderToken name="z-floating" value="50" />
    <OrderToken name="z-overlay" value="100" />
  </div>
</Preview>
