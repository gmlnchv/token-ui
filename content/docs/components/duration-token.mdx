## Overview

**DurationToken** is a specialized variant of the Token component that adds a visual icon to help distinguish motion timing values while preserving the familiar tooltip for inspecting the exact duration.

This component implements the `duration` type from the [W3C Design Tokens specification](https://www.designtokens.org/tr/2025.10/format/#duration) (ยง8.5), which represents time values with units like `ms` or `s`.
