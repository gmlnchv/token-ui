## Overview

**BaseIconToken** is a small helper built on top of `Token` that slots an icon before the token label. All the tooltip behavior and keyboard interactions come from the underlying `Token`, so you can quickly create new icon-flavored variants by passing any icon component.

## Installation

<Installer packageName="icon-token" />

## Usage

Pass any icon component (Lucide or your own) via the `icon` prop. The icon is rendered inline before the label, and the value shows in the tooltip.

<Preview code={`import { BaseIconToken } from '@/components/icon-token'
import { Sparkles } from 'lucide-react'

  <div className="flex gap-2 flex-wrap">
    <BaseIconToken name="special" value="123" icon={Sparkles} />
    <BaseIconToken
      name="custom"
      value="--my-token"
      icon={Sparkles}
      iconClassName="size-5 text-amber-500"
    />
  </div>`}>
  <IconTokenDemo />
</Preview>
