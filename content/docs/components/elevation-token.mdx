## Overview

**ElevationToken** is a specialized variant of the Token component that adds a visual icon to help distinguish elevation and shadow values while keeping the same tooltip interaction for inspecting values.

This component implements the `shadow` composite type from the [W3C Design Tokens specification](https://www.designtokens.org/tr/2025.10/format/#shadow) (ยง9.6), which represents shadow styles with color, offsets, blur, and spread properties.
