import { ElevationToken } from '@/registry/components/elevation-token'

## Overview

**ElevationToken** adds a layers icon next to the token name so elevation and shadow values are easy to spot while keeping the same tooltip interaction for inspecting values.

## Installation

<Installer packageName="elevation-token" />

## Usage

Use `ElevationToken` for shadow or depth values:

<Preview code={`import { ElevationToken } from '@/components/elevation-token'

    <ElevationToken name="shadow-sm" value="0px 2px 8px rgba(15, 23, 42, 0.14)" />
    <ElevationToken name="shadow-md" value="0px 6px 16px rgba(15, 23, 42, 0.18)" />
    <ElevationToken name="shadow-lg" value="0px 12px 32px rgba(15, 23, 42, 0.24)" />
